# Bootstrap

Use this reference for Bootstrap, React-Bootstrap, BootstrapVue, NgBootstrap, or projects organized around Bootstrap variables, grid, utilities, and components.

Prefer Bootstrap's component, spacing, display, grid, and form conventions before custom CSS. Customize through existing Sass variables, CSS variables, theme overrides, or wrapper components rather than scattered one-off selectors.

Keep accessibility attributes aligned with Bootstrap's interactive patterns, especially modals, dropdowns, navs, accordions, toasts, and validation feedback. Do not mix another utility framework into a Bootstrap surface unless the repository already does.
